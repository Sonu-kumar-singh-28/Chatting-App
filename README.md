Chatting-app Readme..

Welcome to the repository for the Chatting-App. This is a real-time mobile messaging application designed to connect users through fast, direct, and seamless communication.

‚ú® Key Features

The application focuses on a smooth and intuitive user experience, offering the following core functionalities:

Real-time Messaging: Instant one-on-one conversations with real-time message delivery.

User Profiles & Management: View and update user details through a dedicated Profile screen.

Secure Authentication: Standard and secure user login and registration flow.

User Search: Find and start conversations with other users by searching usernames.

Chat History (Home Screen): Displays recent conversations with last message preview and timestamps.

Simple Navigation: Bottom navigation for quick access to Home (Chats) and Profile sections.

Logout Functionality: Secure and straightforward logout option.

Screenshots of the app flow are included in the repository for reference.

üöÄ Getting Started

Follow the steps below to set up the project locally.

Prerequisites

Java Development Kit (JDK) 11 or higher

Android Studio

Git

Gradle

Installation & Setup

Clone the repository

git clone https://github.com/Sonu-Kumar-Singh-28/Chat-App.git
cd Chat-App

Firebase Configuration

Create a Firebase project from the Firebase Console.

Download the google-services.json file.

Place it inside the Android app module (app/ directory).

Run Backend (Spring Boot)

Open the backend module in your IDE.

Run the Spring Boot application using Gradle or IDE run options.

Run Android Application

Open the project in Android Studio.

Select an emulator or physical device.

Build and run the application.
